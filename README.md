# Personal Expense Tracker with Data Visualization

## 📌 Project Overview
A Python-based personal expense tracker that generates insights and visualizations from expense data.  
It supports CSV-based tracking, automated analysis, and an interactive Streamlit dashboard with downloadable reports.

---

## ✅ Problem Statement
Manual expense tracking is time‑consuming and does not provide clear insights.  
This project automates tracking, analysis, and visualization to help improve budgeting and spending control.

---

## 🎯 Industry Relevance
Useful for:
- Python Developer roles  
- Data Analyst roles  
- Business Analyst roles  
- Finance / Personal budgeting roles  
- Automation projects  

---

## 🚀 Features
- Synthetic expense data generation
- Category-wise expense analysis
- Monthly spending trends
- Payment method distribution
- Daily spending trend
- Streamlit interactive dashboard
- Downloadable CSV report
- Excel summary report with auto charts

## ⚙️ Installation
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate

pip install -r requirements.txt
```

---

## ▶️ Run Project (Generate Data + Charts)
```bash
python main.py
```

Outputs:
- `data/expenses.csv`
- `outputs/*.png`
- `reports/summary_report.csv`

## 📊 Streamlit Dashboard


- Monthly trend analysis

- Daily spending trend

- Excel report with auto charts
